# BMI Calculator

A simple BMI (Body Mass Index) calculator application built with Python and Tkinter.

## Features
- Calculate BMI based on weight (kg) and height (m)
- Display BMI category (Underweight, Normal weight, Overweight, Obese)
- User-friendly GUI interface
- Input validation

## Requirements
- Python 3.x
- tkinter (usually included with Python)

## How to Use
1. Run the application:
   ```bash
   python main.py
   ```
2. Enter your weight in kilograms
3. Enter your height in meters
4. Click "Calculate BMI" button
5. View your BMI result and category

## BMI Categories
- Underweight: BMI < 18.5
- Normal weight: 18.5 ≤ BMI < 25
- Overweight: 25 ≤ BMI < 30
- Obese: BMI ≥ 30
