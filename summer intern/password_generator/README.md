# Password Generator

A secure password generator application built with Python and Tkinter.

## Features
- Generate random passwords with customizable length
- Choose character types:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Digits (0-9)
  - Special characters (!@#$%^&*)
- Copy generated password to clipboard
- User-friendly GUI interface
- Input validation

## Requirements
- Python 3.x
- tkinter (usually included with Python)

## How to Use
1. Run the application:
   ```bash
   python "import tkinter as tk.py"
   ```
2. Enter desired password length (minimum 4 characters)
3. Select which character types to include
4. Click "Generate Password" button
5. Copy the generated password to clipboard if needed

## Default Settings
- Password Length: 12 characters
- All character types enabled by default
