# 🤖 Nova / Jarvis-Inspired Voice Assistant

A futuristic **voice-controlled AI assistant** built using Python that can perform tasks like opening applications, playing music, answering queries, and interacting through a sleek GUI — inspired by **Jarvis**.

---

## 🚀 Features

* 🎙️ **Voice Recognition** – Understands real-time voice commands
* 🔊 **Text-to-Speech** – Responds with natural voice output
* 🎵 **Play Music** – Plays songs directly from YouTube
* 💻 **Open Applications** – Launch apps like Calculator instantly
* 🌐 **Web Search** – Performs Google searches via voice
* 🧠 **AI Responses** – Handles general queries intelligently
* 💬 **Interactive GUI** – Displays live conversation (Tkinter-based)
* 🔁 **Memory Feature** – Remembers last played song

---

## 🛠️ Technologies Used

* **Python** – Core programming language
* **SpeechRecognition** – Converts voice to text
* **Pyttsx3** – Offline text-to-speech engine
* **Tkinter** – GUI development
* **PyWhatKit** – YouTube automation
* **Subprocess / OS** – System-level operations
* **OpenAI API (optional)** – For AI-based responses

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/voice-assistant.git
cd voice-assistant
```

### 2. Install dependencies

```bash
pip install pyttsx3 SpeechRecognition pywhatkit pyaudio openai
```

> ⚠️ Note: Installing `pyaudio` may require a pre-built wheel depending on your system.

---

## ▶️ Usage

Run the assistant:

```bash
python main.py
```

### 🎤 Example Commands

* "Open calculator"
* "Play Believer song"
* "Play it again"
* "Search Python tutorials"
* "Open YouTube"

---

## 🧠 How It Works

1. Captures voice input using microphone
2. Converts speech → text using SpeechRecognition
3. Processes command using Python logic
4. Executes action (open app, play song, etc.)
5. Responds via Pyttsx3 voice output
6. Displays interaction in GUI
